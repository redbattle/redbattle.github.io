> 尽量使用服务器搭建，不使用电脑本机搭建

## 安装环境

- 安装 1panel，如果下方命令错误可以去 [官网](https://1panel.cn/) 或者 [GitHub](https://github.com/1Panel-dev/1Panel) 使用新命令

  ```
  bash -c "$(curl -sSL https://resource.fit2cloud.com/1panel/package/v2/quick_start.sh)"
  ```

- 安装好 1panel 记下面板信息，在云服务器平台设置服务器安全组，在浏览器中打开面板地址进行登录

- 打开左侧菜单 **应用商店**，安装 `ollama`、`OpenClaw`

- 打开左侧菜单 **面板设置** -> **默认访问地址**，地址设置成 **服务器外网 IP**

## 安装模型

- 打开左侧菜单 **AI** -> **模型**，添加 Ollama 模型，

- 填写名称，此处名称即为模型 ID，可前往 [Ollama 官网](https://ollama.com/search) 查看

- 安装完模型后，点击列表的 **运行** 操作项，在打开的终端里输入文字，看是否能正常对话

## 创建智能体

- 打开左侧菜单 **AI** -> **智能体**，创建 **模型账号**

  - **模型供应商** 选 `Ollama`
  - **API Key** 本地部署 AI 模型用不到，可以填任意值；远程调用 `token` 填各自平台的 API Key
  - **Base URL** 填 Ollama 容器的内网 IP，格式 `http://<Ollama容器IP>:11434/v1`

- 打开左侧菜单 **AI** -> **智能体**，创建 **智能体**
  - **模型供应商** 选 `Ollama`
  - **模型账号** 选上一步创建的账号
  - 勾选 **手动输入模型**
  - **模型** 填写安装的模型，按照 `Ollama/模型ID` 的格式填写
- 打开 `openclaw` 容器配置文件 `openclaw.json`，修改 ollama 配置
  ```
  "models": {
    "mode": "merge",
    "providers": {
      "ollama": {
        "api": "openai-completions",
        "apiKey": "ollama",
        "baseUrl": "http://<Ollama容器IP>:11434/v1",
        ...
      }
    }
  }
  ```

## 使用

- 创建 **智能体** 后，点击列表 `WebUI 端口: xxx` 可直接打开浏览器访问
- 如果项界面更美观可安装 `OpenWebUI` 体验
