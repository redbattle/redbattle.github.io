Tailwind

  • 安装依赖

    npm install -D tailwindcss postcss autoprefixer

  • 添加 Tailwind 到 PostCSS 配置

    // postcss.config.js
module.exports = {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
    ...(process.env.NODE_ENV === "production" ? { cssnano: {} } : {}),
  },
};

  • 配置 Tailwind

    // tailwind.config.js
module.exports = {
  content: ["./src/**/*.{html,js}"],
  theme: {
    extend: {},
  },
  variants: {},
  plugins: [],
};

  • 添加 Tailwind 到 main.js 或者 main.css

    // mian.js
import "tailwindcss/tailwind.css";

    /** mian.css */
@tailwind base;
@tailwind components;
@tailwind utilities;
